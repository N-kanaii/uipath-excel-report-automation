# uipath-excel-report-automation
帳票作成業務の効率化を目的として開発したUiPathロボットです。 外部設定ファイルによる柔軟なパス管理とDictionaryを用いたデータ管理を行い、複数データを1件ずつ処理してExcel帳票へ自動転記します。  業務要件に基づき、条件分岐処理や和暦変換を実装しており、実務運用を想定した設計となっています。 本リポジトリは機密情報を除去したポートフォリオ用サンプルです。


前提条件
請求ソフトへログインした状態で稼働


主な機能
外部設定ファイルの読み込み（パス・各種設定の管理）
一覧データの1件ずつ処理
集計したデータを計算
Excel帳票フォーマットへの自動転記
担当者情報・日付の動的反映
和暦変換処理
ログ出力による処理状況の可視化

処理フロー
Main.xaml
 ↓
Setdata.xaml（外部設定ファイル読み込み）
 ↓
stationlist_read.xaml（一覧データ取得）
 ↓
original_writting.xaml（帳票へ書き込み）
 ↓
完了

使用技術
UiPath（RPA）
Excel Automation
Dictionaryによるデータ管理
DataTable処理
条件分岐（If）
ログ出力

ポイント
外部設定ファイルを利用することで、パス変更や環境差分に柔軟に対応
Dictionaryを用いたデータ管理により、可読性と拡張性を向上
処理を複数ワークフローに分割し、保守性を意識した設計
業務帳票仕様に合わせた和暦変換処理の実装
